
# 🐾 Pawfect Match

Pawfect Match is a full-stack pet adoption platform developed as part of a university Software Engineering project. The platform provides users with a modern and responsive experience to browse adoptable pets, explore adoption plans, manage personal accounts, and interact with the system through a clean and intuitive interface.

---

## 🚀 Features

- User authentication
- Pet browsing
- Adoption plans
- User profiles
- Responsive user interface
- Session management
- Database integration
- Dynamic web pages using EJS

---

### Frontend

- CSS3

- EJS

- Node.js

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/joharaaf/pawfect-match.git
Go to the project folder

```bash
cd pawfect-match
```

Install all dependencies

```bash
npm install
```

Create a `.env` file (if required by your project) and configure your database credentials.

Example:

```env
DB_HOST=localhost
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=your_database
```

Start the application

```bash
npm start
```

or

```bash
node app.js
```

The application will then be available on:

```
http://localhost:3000
```
